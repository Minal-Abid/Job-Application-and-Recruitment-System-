# Job Application System Database

## Overview
This project is a complete database design for a Job Application and Recruitment System developed as part of a Database Management Systems course.

The system manages:
- Applicants
- Recruiters
- Job postings
- Applications
- Hiring workflow

## Project Workflow
1. Requirement Analysis
2. ERD Design
3. Relational Schema Mapping
4. SQL Implementation
5. Query Testing

## Technologies Used
- MySQL
- SQL
- Database Design Concepts

## Project Structure
ERD/ → Entity Relationship Diagram  
Schema/ → Relational schema design  
SQL/ → Database creation and queries  
Documentation/ → Project report

## How to Run
1. Open MySQL Workbench
2. Run `create_tables.sql`
3. Insert data using `insert_data.sql`
4. Execute queries using `queries.sql`

OR

Run `full_workflow.sql` to execute the complete setup.
